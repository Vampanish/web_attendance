Roll Call Management System

About the Project

We are developing a student attendance management system for our college. This project is undertaken by a team of three members: Anish, Manikandan, and Solomon Prince. The system will allow professors to mark and manage student attendance efficiently.

Database Used

We are using MySQL as our database. MySQL is a popular open-source relational database management system that allows efficient storage and retrieval of data. In this project, MySQL is used to store student records, attendance data, and related information.

HTML Tags Used

Below are some of the HTML tags used in creating this website:

  • <header>: Defines the header of the page.
  • <p>: Used for paragraphs.
  • <h1>, <h2>, <h3>: Defines headings with different levels.
  • <ul> and <li>: Used to create unordered lists and list items.
  • <strong>: Makes text bold for emphasis.
  • <table>, <tr>, <th>, <td>: Creates tables for structured data.
  • <img>: Inserts images.
  • <a>: Defines hyperlinks.
  • <blockquote>: Highlights quotes or important text.
  • <code>: Represents code snippets.
  • <style>: Adds CSS styling within the HTML file.
